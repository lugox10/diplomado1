# Semana 3 - QA Asincrono, TestOps y CI/CD

Esta semana contiene el material de clase de los dias 4 y 5.
La idea es practicar con una app sencilla de laboratorio, no con una app de produccion.

## Estructura

- `dia-4-kafka-docker/`
  - App de clase para entender procesamiento asincrono con Kafka.
  - Incluye backend (`app`), frontend (`frontend`) y pruebas E2E (`tests`).
- `dia-5-testops-cicd/`
  - Ejercicio de clase para ejecutar el flujo en CI/CD con GitHub Actions.

## Orden recomendado para alumnos

1. Inicia por `dia-4-kafka-docker` y ejecuta el flujo local completo.
2. Luego pasa a `dia-5-testops-cicd` para ver el mismo flujo en pipeline.

## Nota de uso

Cada carpeta tiene su propio README con pasos concretos.
