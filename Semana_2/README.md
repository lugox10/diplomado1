# Semana 2 - QA Multi Framework

Esta semana contiene proyectos independientes para practicar QA con Cypress, Playwright y Selenium.

## Requisitos

- Node.js 18+
- Java 17+
- Maven 3.9+
- PowerShell (Windows)

## Orden sugerido para alumnos

1. API primero (más fácil):
   - Cypress API
   - Playwright JS/TS API
   - Selenium API (Maven o Clean)
2. E2E después:
   - Cypress E2E
   - Playwright JS/TS E2E
   - Selenium E2E
3. Prácticas:
   - practica/playwright
   - practica/java/demo

## Ejecución rápida por lotes

Desde esta carpeta puedes usar:

```powershell
.\validar-pokeapi-tests.ps1
.\ejecutar-todo-pokeapi.ps1
.\ejecutar-todo.ps1
```
